# Etch-a-Sketch

A browser-based sketchpad project built as part of The Odin Project curriculum. It lets you draw on a dynamic grid by hovering over squares, mimicking the classic Etch-a-Sketch toy.

## Features

- **Dynamic Grid Size:** Use a slider to change the grid resolution up to 100x100.
- **Customizable Colors:** Choose a specific color using the color picker.
- **Rainbow Mode:** Cycles through the colors of the rainbow sequentially as you draw.
- **Progressive Darkening:** Each pass over a cell increases its opacity by 10%, fully coloring it in 10 passes.
- **Auto Fill:** A one-click button to instantly apply the current color settings to every cell on the board.
- **Clear Canvas:** Easily erase your masterpiece and start fresh.

## Technologies Used

- HTML
- CSS (Flexbox)
- JavaScript (DOM manipulation)
